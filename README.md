# Multi-Room Hidden-Camera Detection Protocol

This repository contains a GitHub Pages site for a multi-room hidden-camera detection user-study protocol. The website is written as a static site in `docs/` using plain HTML, CSS, and a small amount of vanilla JavaScript.

## Publish With GitHub Pages

1. Commit the files in this repository.
2. Go to the repository **Settings** page.
3. Open **Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and the `/docs` folder.
6. Save the settings.

GitHub Pages will publish the content from `/docs`.

## Local Preview

Open `docs/index.html` in a browser. No build step or package manager is required.
