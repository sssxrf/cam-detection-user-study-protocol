Participant checklist

User Flow Checklist

Use this checklist to follow the app workflow from setup to the end of the session.

Read Instructions Back Home

Choose Method

  • Open the study app.
  • Choose AR + Button, QR Code, or NFC Tag.
  • Read the short instructions shown by the app.
  • Confirm that the connected sniffer is ready if the app displays a sniffer status check.

Room Setup

AR + Button

  • Confirm the room map or room list shown by the app.
  • Confirm the starting location if the app asks.
  • Continue to Start Detection.

QR Code

  • Enter the number of rooms.
  • Place or select one QR code near each room entrance.
  • Scan each QR code.
  • Name each room.
  • Confirm each room-to-code mapping.
  • Review the final room table.

NFC Tag

  • Enter the number of rooms.
  • Place or select one NFC tag near each room entrance.
  • Tap each NFC tag.
  • Name each room.
  • Confirm each room-to-tag mapping.
  • Review the final room table.

Start Detection

  • Tap Start Detection only after room setup is complete.
  • Confirm that the app starts the guided session.
  • Confirm that the app starts network capture through the connected sniffer.
  • Do not operate the sniffer separately.

For Each Room Action

  • Read the room name shown by the app.
  • Stand outside the target room.
  • Mark the start of the action: AR: tap Start Action. QR: scan the room QR code. NFC: tap the room NFC tag.
  • Enter the room normally.
  • Walk naturally along the instructed path.
  • Exit the room.
  • Mark the end of the action: AR: tap Action Finished. QR: scan the same QR code again. NFC: tap the same NFC tag again.
  • Wait for the next app prompt.

Mistake Checklist

  • If I used the wrong room, tap Report Mistake.
  • If I used the wrong tag, tap Report Mistake.
  • If I forgot a scan, tap, or button press, tap Report Mistake.
  • If QR/NFC/AR fails, tap Report Mistake.
  • Wait for the app to restart the affected step.

Finish Session

  • Follow the app's final prompt.
  • Let the app stop the guided session.
  • Let the app stop network capture automatically.
  • Do not manually edit the recorded data.
