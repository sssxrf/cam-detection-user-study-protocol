Participant checklist

User Flow Checklist

Use this checklist to follow the app workflow from area setup to grouped detection rounds and the end of the session.

Read Instructions Back Home

Choose Method

  • Open the study app.
  • Choose AR + Button, QR Code, or NFC Tag.
  • Read the short instructions shown by the app.
  • Confirm that the connected sniffer is ready if the app displays a sniffer status check.

Area Setup

AR + Button

  • Confirm the room/area map or area list shown by the app.
  • Confirm the starting location if the app asks.
  • Continue to connection setup.

QR Code

  • Enter the number of areas.
  • Place or select one QR code near each area entrance.
  • Scan each QR code.
  • Name each area.
  • Confirm each area-to-code mapping.
  • Review the final area table.

NFC Tag

  • Enter the number of areas.
  • Place or select one NFC tag near each area entrance.
  • Tap each NFC tag.
  • Name each area.
  • Confirm each area-to-tag mapping.
  • Review the final area table.
  • I named all rooms/areas.
  • I included hallways and connected spaces if I may pass through them.

Connection Setup

  • I marked which areas are connected.
  • I marked areas that share doors, open boundaries, or hallways.
  • I marked pass-through areas.
  • I marked areas that must be passed through to reach another area.
  • I confirmed the connection graph.

Grouped Detection Rounds

  • I reviewed the app-generated target groups.
  • I understand that the app may use two groups or three or more groups.
  • I understand that I will test one group at a time.
  • I understand that guard/transit areas are not judged as clean non-target areas in that round.

Start Detection

  • Tap Start Detection only after area setup, connection setup, and group review are complete.
  • Confirm that the app starts the guided session.
  • Confirm that the app starts network capture through the connected sniffer.
  • Do not operate the sniffer separately.

For Each Target-Area Action

  • Read the area name shown by the app.
  • Stand outside the target area.
  • Mark the start of the action: AR: tap Start Action. QR: scan the area QR code. NFC: tap the area NFC tag.
  • Enter the target area normally.
  • Walk naturally along the instructed path.
  • Exit the target area.
  • Mark the end of the action: AR: tap Action Finished. QR: scan the same QR code again. NFC: tap the same NFC tag again.
  • If the route requires passing through another area, follow the app prompt and do not count that as a target-area action unless the app asks.
  • Wait for the next app prompt.

Mistake Checklist

  • If I used the wrong area, tap Report Mistake.
  • If I used the wrong tag, tap Report Mistake.
  • If I forgot a scan, tap, or button press, tap Report Mistake.
  • If I followed the wrong route or entered an extra area, tap Report Mistake.
  • If QR/NFC/AR fails, tap Report Mistake.
  • Wait for the app to restart the affected step.

Finish Session

  • Follow the app's final prompt.
  • Let the app stop the guided session.
  • Let the app stop network capture automatically.
  • Do not manually edit the recorded data.
